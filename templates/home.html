Welcome to LoremChain

What is LoremChain?

LoremChain is a decentralized, peer-to-peer digital transaction network based on the same Blockchain Technology as Bitcoin and Ethereum.

Blockchain Technology?

A Blockchain is a list of blocks, where each block contains some data and a unique link to the previous block in the chain.

What does that mean?

Once a block is added to the chain, it is linked with the previous chain in such a way that if block data is then changed, the link breaks.

What good is that?

Since trying to change data on the chain will break the links, once data is in a block it can't be changed without alerting the network.
